# SNAKES-LADDERS-GAME-
🎲 Snakes and Ladders in C  
A simple console-based Snakes and Ladders game built in C. It features two-player gameplay, random dice rolls, and classic board mechanics with snakes and ladders. This project demonstrates basic programming concepts like loops, conditionals, arrays, and game logic.

📖 About the Project
This project is designed to demonstrate fundamental programming concepts through a fun and interactive game. It helps beginners understand how to implement logic, control flow, and randomness in C.

---

🚀 Features
- 🎮 Two-player gameplay
- 🎲 Random dice generation using `rand()`
- 🐍 Snakes that move players backward
- 🪜 Ladders that move players forward
- 🔁 Turn-based player switching
- 🏆 Winning condition detection
- 📟 Simple console interface

---

🎯 Game Rules
1. The game is played between two players.
2. Each player rolls a dice (1–6) on their turn.
3. Players move forward based on the dice value.
4. Landing on a ladder moves the player up.
5. Landing on a snake moves the player down.
6. The first player to reach position 100 wins.
7. Exact roll may be required to win (depending on implementation).
